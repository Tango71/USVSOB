All bot commands are logged.(DDDS)

as a member, you are only authorized to use the commands listed in the !help area.
anyone caught using a command that they are not authorized to use will need to explain to CJ Or Adda why the command was used without authorization.

The SOB contains a list of all the SOP's, as well as game status images.

a list of commands can be found by typing !help

as the bot is still new. please let Tango71 know if there is an error. and it will get fixed.

this bot is up 24/7 
If for some reason the bot commands are not working, please inform @commanders to check the bot status in the #Intel channel.

once you give the !help command the bot will send you a PM with the info you requested. you can run all sop commands from that PM.

zombie kill of the day will need to be run from within the #drill-hall channel by typing !z8d or !z7d thru z1d. this will automatically inform commanders

Academy Up status can also be issued by typing !ao the bot will automatically send it to everyone.
the complete list of commands can be found below.
